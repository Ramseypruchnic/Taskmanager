💡 How to Use Gmail Integration:

  1. Connect Gmail: Click "Connect Gmail Account" and authorize access
  2. Set Keywords: Add keywords like "TASK", "TODO", "PROJECT" to filter emails
  3. Send Emails: Email yourself with keywords in subject: "TASK: @strategylabs @high Call client"
  4. Import Tasks: Click "Import Tasks from Gmail" to automatically create tasks
  5. Use @ Syntax: Include @category and @priority in email body for auto-categorization

📧 Email Format Examples:

Subject: TASK: Client Follow-up
Body: @strategylabs @high Call client about proposal deadline
Subject: TODO: Weekend Tasks
Body: @personal @low Buy groceries and clean garage
